Answer

Layout primitives

Each card lives in a rounded dark panel with a small uppercase header (icon + label) separated from the body by a thin border. Cards stack vertically with consistent spacing. Long content scrolls inside the iframe; the sidebar to the left stays fixed.

Prose styling

Paragraphs use the Inter font at 15 px with a generous line height. Bold and italic emphasis read clearly against the dark background, and inline links use the royal-blue accent with a dashed underline that solidifies on hover.

  • Unordered lists indent with a hanging bullet
  • Multiple items get consistent vertical rhythm
  • Nested content stays readable
  1. Ordered lists use the same spacing
  2. The numbers inherit the body color

Inline code and callouts

Inline code like const foo = bar() uses JetBrains Mono in a subtle pill. The .callout helper highlights short emphasized blocks inside the body:

This is what a callout looks like — useful for "do this" quotes, recommended replies, or any short emphasized text that needs to stand out from the surrounding prose.
